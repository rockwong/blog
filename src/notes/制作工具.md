# 制作工具

## TinyPNG 图片压缩

###node-tinypng工具

```bash
	npm install node-tinypng -g
```
命令行中进入图片目录 运行命令：

```bash
	tinypng -k 1PAcFiAypEgL19cLkbZpd6VbnnOyeckH *.png
```
其中 1PAcFiAypEgL19cLkbZpd6VbnnOyeckH 是在tinypng网站注册的key，每月可以压缩500张图片。如果用完，可以再注册一个key

> 参考 https://www.npmjs.com/package/node-tinypng
